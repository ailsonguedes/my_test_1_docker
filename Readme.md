# My Test 1 (Docker)

Esse programa foi desenvolvido como um primeiro teste de utilização do Docker.


## 🔧 Ferramentas Utilizadas

-   Docker:  plataforma de contêineres que permite que os desenvolvedores empacotem seus aplicativos em contêineres portáteis que podem ser executados em qualquer ambiente. Isso torna a implantação e a execução de aplicativos muito mais fáceis e eficientes, pois as dependências e o ambiente de execução são encapsulados no contêiner.

## ⚙️ Como Funciona

-   Ao rodar a aplicação o usuário deverá receber a mensagem "Olá, Docker!".

## 💻 Como executar o código

- Instalar Python 
- Abra o prompt de comando
- caminhe até a pasta raiz do programa e contém os arquivo "app.py" e "Dockerfile".
- Você precisa inicialmente criar uma imagem Docker a partir do Dockerfile. Utilize o seguinte comando "docker build -t meu-aplicativo-python .".
- Para executar o arquivo  (**docker run meu-aplicativo-python**)
